# ractive-engine
#### Template engine for node.js with Ractive.js.

Features like
- extend/block in jade or
- layout/renderBody section/renderSection in ASP.NET MVC Razor

are supported. Actually they are the same thing.

# Usage
## extend
### syntax
```html
{{#extend someLayout}}
{{/extend}}
```

## block
### syntax
```html
{{#block body}}
{{/block}}
```
